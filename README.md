# TaskList

Uma aplicação de **lista de tarefas (Todo App)** desenvolvida com **React** e focada em boas práticas, UI moderna e código limpo. O projeto permite criar, concluir e remover tarefas, além de apresentar estatísticas visuais sobre o progresso.

---

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Estatísticas de tarefas concluídas e pendentes
- Gráfico de pizza para visualização do progresso
- Exibição de data e hora de criação da tarefa
- Notificações visuais (toast)
- Interface moderna com Tailwind CSS

---

## Tecnologias Utilizadas

- **React** (Vite)
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **Recharts** (gráficos)
- **Sonner** (notificações)
- **Font Awesome** (ícones)
- **Git & GitHub** (versionamento)

## Como executar o projeto localmente

1. Clone o repositório:

```bash
git clone https://github.com/caolodev/TaskList.git
```

2. Acesse a pasta do projeto:

```bash
cd TaskList
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

5. Abra no navegador:

```
http://localhost:5173
```
